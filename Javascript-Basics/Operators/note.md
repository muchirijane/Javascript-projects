### Operator - transfer values or combine values

## Types of Operators in JavaScript

### 1. Arithmetic Operators - used to do sums in javascript

| Operator | Description                  |
| -------- | ---------------------------- |
| +        | Addition                     |
| -        | Subtraction                  |
| \*       | Multiplication               |
| \*\*     | Exponentiation               |
| /        | Division                     |
| %        | Modulus (Division Remainder) |
| ++       | Increment                    |
| --       | Decrement                    |

### 2. Assignment Operator

Example assign

```js
let score = 10;
score += 50;
console.log(score); // 60

//This is the same as
let score = 10;
score = score + 50;
console.log(score); // 60
```

They assign value to a variable.
